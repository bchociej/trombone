# trombone!

Play a trombone sound!


## Programmatically!

```javascript
var trombone = require('trombone');
trombone.play();
```

## From command line!

```bash
$ sudo npm install -g trombone
$ trombone
```

## As an npm script!

Add "trombone" to your dependencies, run `$ npm install`, then define an npm script as `node_modules/.bin/trombone`!
